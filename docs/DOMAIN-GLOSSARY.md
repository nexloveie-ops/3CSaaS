# Domain glossary (EN / 中文)

| English | 中文 | Notes |
|---------|------|-------|
| Company | 公司 / 法律实体 | Tenant root, subscription, tax, products |
| Store | 店铺 | Stock location, POS, reports |
| Chain | 连锁（虚拟） | Cross-company network; paid module |
| Warehouse | 仓库 | Store + `warehouse` module; B2B only outward |
| Serial unit | 序列号实例 | One SN/IMEI per document |
| Transfer order | 内部调拨单 | Same company; cost price; no VAT invoice |
| B2B order | B2B 订单 | Cross-company; seller invoice |
| Work order | 维修工单 | Tied to serial unit |
| Pre-order | 预订 | Deposit → convert or cancel + credit note |
| Credit note | 贷项单 | Refund / reversal |
| Margin VAT | 差额增值税 | Second-hand; dual invoice perspective |
| Daily summary | 日结汇总 | Statistics only, no edit lock |
| Stock share rule | 库存共享规则 | Quantity or percent visible to chain |
